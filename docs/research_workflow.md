# 研究工作流

Analysis Workspace 默认使用中文，支持切换英文。研究分析页展示数据审计、事件研究、因子分箱、因子 IC、候选规则、滚动样本外验证和 Markdown 报告。

研究页可选择目标标签，默认使用 `fwd_ret_10_side_adj`。事件研究按样本数排序，因子 IC 按 `abs(spearman_rank_ic)` 排序，候选规则优先按测试期评分排序。排序和筛选用于查阅结果，不改变“候选规则不是交易信号”的边界。

## 两类问题不能混写

Research Analysis 回答已标记事件的问题：标签后的结果分布、可解释因子与目标标签关系、候选规则在后续样本中的衰减。

Time-Series Diagnostics 回答行情序列的问题：对数收益分布、自相关、波动聚集、尾部风险、短周期噪声 proxy，以及多币种共同因子。

行情存在厚尾或波动聚集，不等于某类交易事件有优势。事件样本表现较好，也不等于未来市场序列可预测。

## 特征和标签

`research.factor_library.FeatureFactory` 只构造事件当根及之前可见的模型输入。未来收益、事后路径、MFE、MAE 和人工最终交易结果只属于标签。

未来函数审计失败时，研究包被标记为无效。候选规则只是待验证假设，不是交易信号。

## 报告与导出

研究报告支持 `language="zh_CN"` 和 `language="en_US"`。导出包保留原始英文字段键，界面和 Markdown 报告按语言显示，以保证 CSV/JSON 下游接口稳定。

所有统计结果都是探索性证据。样本内结果不代表未来表现，多重检验会增加假阳性风险，本工具不构成投资建议，也不连接实盘下单 API。
