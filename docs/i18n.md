# 界面语言

研究工作台使用 `quant_collector_app/i18n.py` 读取 `translations/zh_CN.json` 和 `translations/en_US.json`。

- 没有显式语言设置时，研究页默认中文。
- 主界面当前语言为 `en_US` 时，研究页 tab、按钮和表头显示英文。
- CSV / JSON 的字段键不翻译，避免导出接口随界面语言变化。
- `research_report.md` 与 `time_series_report.md` 均支持 `language="zh_CN"` 或 `language="en_US"`。

新增可见文本应加入翻译资源，不应散落写死在分析窗口代码中。
